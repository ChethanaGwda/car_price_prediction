# 🚗 Car Price Prediction System

This project predicts the selling price of a used car using Machine Learning.
The model analyzes car details like:

- Manufacturing Year  
- Present Price  
- Kilometers Driven  
- Fuel Type  
- Seller Type  
- Transmission  
- Number of Owners  

and gives an estimated resale price of the car.

---

## 📊 Project Overview

The system uses **3 Machine Learning models**:

- Linear Regression  
- Lasso Regression  
- Random Forest Regression (**Best Model**)  

**Random Forest** gives better accuracy, so it is used for the final prediction.
